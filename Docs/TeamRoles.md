## Collaboration Method

- The team uses GitHub for version control.
- Each feature is developed in a separate branch.
- Pull Requests are reviewed before merging into main.
- GitHub Projects is used to track tasks and progress.
- Daily short meetings are held to discuss completed work, blockers, and next tasks.
